# Creating a resource group
{: #create-resource-group}

Following are the steps to create a resource group:

1. In the IBM Cloud console, click **Manage** > **Account**.

2. In the **Account** navigation menu, click **Account resources** > **Resource groups**.

3. Click **Create** to create a new resource group.

4. Enter the name for the resource group (for example, "sandbox-resources").

5. Optionally, add a **Description** to help identify the purpose of this resource group.

6. Click **Create** to finalize the resource group creation.

The new resource group will now appear in your resource groups list and can be selected when creating a sandbox environment.
